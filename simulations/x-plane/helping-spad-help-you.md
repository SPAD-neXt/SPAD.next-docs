# Spad.neXt Gets its Type Rating

Good news, The latest version of **SPAD.neXt** no longer needs you to play file-renaming ground crew. It now automatically discovers **all X-Plane datarefs and commands** on its own.

That’s right. No more generating last-run files.\
No more renaming text files like you’re hacking NORAD.\
No more dragging mysterious `.txt` files into aircraft folders hoping you didn’t break something.

SPAD.neXt now scans the aircraft and sim environment directly and builds its own database like a well-trained avionics suite.

In aviation terms?\
We’ve upgraded from paper charts to full glass cockpit.<br>

Let’s be crystal clear:

❌ You no longer need:

* `datarefs.txt`
* `commands.txt`

✅ You now:

* Open SPAD.neXt
* Select your aircraft
* Search for any dataref or command directly inside SPAD.neXt
* Assign it
* Fly happily ever after

No pre-flight file rituals required.
